Настройки плагина WGAPIAuth

{ALERT}

Если этот параметр отмечен, настройки wordpress ("Любой может зарегистрироваться") игнорируются, новый аккаунт будет создан.

Эсли заполнить поле, ссылка для входа появится в форме авторизации. После аутентификации пользователь будет перенаправлен на главную страницу сайта.

Если необходимо, вы можете создать свой html елемент (ссылку, кнопку, картинку...) в нужной части сайта, отправляйте пользователя по адресу {AUTH_URL} что бы авторизировать его. После аутентификации пользователь будет перенаправлен на страницу сайта, на которой он кликнул по ссылке, если GET параметр rt не задан. В качестве значения GET параметра можно указать ссылку, тогда пользователь, после аутентификации будет перенаправлен по ней, если ссылка ведет на ваш сайт. Если значение параметра пустое, пользователь попадет на главную страницу.

Эсли заполнить поле, ссылка для входа появится в форме добавления комментариев, если пользователь еще не аутентифицирован.
